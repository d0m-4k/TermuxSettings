## TermuxSettings
Начальная настройка для Termux с баннерами и темами

Настройка содержит:
* Termux Banner от Bhai4You
* Termux Ohmyzsh от Cabbagec
* Установка начальных библиотек:
   * python
   * nano
   * wget
   * git
   * и тд.
 
### Установка
```bash
git clone https://github.com/d0m-4k/TermuxSettings
cd TermuxSettings
bash install.sh
```
